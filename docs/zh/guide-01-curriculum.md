# 学习指南 01 — 海外学员的 4 个月学习计划如何运转

> 第一讲(视频)的延伸阅读。这份指南把视频里的动机转换成一份实际可执行的学习计划。
> 写给中文母语者:这份指南本身不是为你写的,但它告诉你 —— 一个海外学员在你眼皮底下,用什么节奏读你写的中文。

---

## 一、海外学员开始之前的 5 个自检

学员在开课前回答的五个问题:

1. 之前学过中文吗?
2. 看到 「推進」「訓練」「部署」 这种汉字词,能不能猜出意思?(韩语母语者通常都行)
3. 一周能投入 5 小时以上吗?
4. 学的是口语还是阅读?
5. 4 个月后愿意用中文写一个真实的 GitHub Issue 吗?

回答模式不同,我们给出的路径也不同。比如 1+2 都是 "是" 的韩语学员,我们走标准 4 个月。1 是 "是" 但 2 是 "否",我们把第一个月的汉字阶段从 4 周拉到 5 周。3 是 "否",直接拉到 6 或 8 个月。4 选 "口语",我们劝他换课程 —— 这门课不是为他设计的。

---

## 二、4 个月路线图(海外学员视角)

| 月份 | 阶段 | 海外学员产出 |
|------|------|--------------|
| Month 1 | 拼音 / 汉字 / 语法骨架 | 累计 300-500 汉字 + 15 个句型 |
| Month 2 | 技术文档阅读 | 5 篇 model card + 10 篇论文摘要 |
| Month 3 | 社区内容阅读 | 知乎答疑 3 篇 + CSDN 文章 2 篇,累计 1000 汉字 |
| Month 4 | 写作实战 | 1 个真实 GitHub Issue + 1 篇知乎回答(300 字+) |

每个月的产出都是看得见的,不是 "我感觉自己进步了" 这种主观估计。Month 4 没交出 1 个 Issue,就算没毕业。

---

## 三、海外学员的日常节奏

**工作日 45 分钟**

```
10 分钟  Anki 复习 30 个汉字 / 词
25 分钟  读一段真实材料 (model card / 知乎 / CSDN)
10 分钟  整理生词、加 Anki、跟 DeepL 对比
```

**周末 90 分钟**

```
30 分钟  Anki 集中清算
45 分钟  通读一整页长文
15 分钟  周回顾:命中率、卡点、下周调整
```

## 四、给中文母语者的三件事

### 1. 你写的中文,是这套学习计划的核心教材

我们不发创作型教材。我们让海外学员去读 你 写的 model card、 你 写的知乎回答、 你 写的 CSDN 文章。也就是说,你写的每一篇中文技术文,都在第二天某个海外学员的 45 分钟阅读时段里出现。这不是夸张。

请尽量保持:
- 术语稳定(同一概念在同一篇里别忽然 「推理」 忽然 「inference」 切换)
- 结构清晰(主谓宾别套太深)
- 段落短(海外学员一段花 5-10 分钟,过长的段他读不完)

### 2. 海外学员看上去会有点 "怪"

他们会:
- 不知道 「你好」 后面接什么,但能流畅写出 「该模型采用 MoE 设计」
- 用 「采用」、「基于」 用得过度,因为这是他刚学的 15 个句型之一
- 偶尔写出语序硬到像翻译机的句子

不是冷淡,不是不礼貌。是阅读优先策略训练出来的语言形态。请理解。

### 3. 反向运用 —— 中文母语者怎么快速读懂英文一手资料

这套方法本身可以反过来用。如果你是中文母语者,想快速读 PyTorch / vLLM / arXiv 一手资料:

- **不要从口语词起步**。don't, gonna, hey 这些跟你想读的英文 README 没关系
- **直接攻领域词汇** —— inference, deployment, quantization, fine-tuning 这些 200 个词
- **直接攻技术写作语法** —— passive voice (被动)、it is + adj. + that(强调)、given that(条件)、in order to(目的)、provided that(条件)、对应中文的 「把/将/用于/基于/对...进行」
- **不读对话练习** —— 直接读 Hacker News、Stack Overflow、PyTorch README

时间节奏可以参考海外学员这边的 4 个月路线图,把 「汉字」 换成 「领域词根 + 词缀」,把 ModelScope 换成 Hugging Face,把知乎换成 lobste.rs。

## 五、毕业产出

海外学员的毕业条件只有两个:

1. 提交一个真实的中文 GitHub Issue(到 vLLM / ms-swift / FastChat / ChatGLM 等中文开源项目)
2. 发一篇 300 字以上的中文知乎回答

如果你看到一个海外开发者发的中文 Issue 或知乎回答,语气有点僵硬,请记得 —— 这是他 4 个月学习路径的最后一步。请你温柔地接住它。

## 六、常见问题(给好奇的中文母语朋友)

**Q. 这套课程对中文母语者有用吗?**
A. 学习材料对你没用。但作为方法论(领域优先 + 阅读优先)可以反向用于英文学习。

**Q. 海外学员真的能在 4 个月学到能读 model card 的水平?**
A. 韩语和日语母语者可以。英语母语者通常需要 6-8 个月,因为没有汉字直觉。

**Q. 我能怎么帮?**
A. 如果你想审阅这门课的语法例句、术语翻译,欢迎提 PR。详见 `CONTRIBUTING.md`。
