# 学习指南 00 — 完全零基础海外学员的起点

> 这份指南是给从来没接触过中文的海外学员看的「0 阶段」。在打开 guide-01 到 guide-04 之前先通读一遍。
> 给中文母语朋友看的版本 —— 这一份特别能让你理解 「外国人学中文,到底是从哪里开始的」。
> 没什么需要背的。30-45 分钟通读建立直觉。

---

## 一、零基础海外学员第一周到底在解决什么?

中文母语者从小到大默认拥有的东西,海外学员从零开始的时候要逐一处理:

1. **汉字是什么:**为什么一个字能直接表达意义,而不是字母拼出来的
2. **简体和繁体:**他要确认 ModelScope 是简体,放心了才敢开始
3. **拼音:**这不是发音课,是 「Pleco 输入法」
4. **声调:**不是发音,是 「输入时要选哪个声调键」
5. **语序:**SVO,跟英文一样,所以英文母语者很快能上手;跟韩日不一样,韩日学员要适应一周

母语者不会意识到这五点是 「需要先解决的问题」,因为对你来说它们是空气。但对学员来说,这就是 0 阶段的全部。

## 二、汉字是什么 —— 给海外学员的解释方式

我们用最直观的例子让海外学员第一次看懂汉字怎么工作:

```
木   一棵树
林   两棵树 = 树林
森   三棵树 = 森林
```

```
日   太阳
月   月亮
明   日 + 月 = 明亮
```

中文母语者读到这里只觉得 「这不是常识吗」,但对海外学员来说,**这是他们第一次理解 「字本身就是意思,不只是声音」**。这个观念一旦建立,他们对剩下几千个汉字就有了正确的心智模型。

### 部首 —— 关键的概念

我们告诉学员:每个汉字大致由 「部首 + 其他」 组成,部首给出意义类别。我们让他们看 10 个最常见的部首:

```
扌(手)  → 推、拉、打、接
氵(水)  → 流、海、河
火     → 烧、炒、烤
心/忄  → 想、思、慢
讠(言) → 说、训、词
```

这一步对海外学员是 「啊!原来汉字是有结构的!」 的瞬间。中文母语者读到这里大概只觉得废话,但这件事在学员的学习曲线上是一个明显的台阶。

### 积木效应

我们给学员看的核心 「卖点」 是这个:

```
学了一个 「推」 →
  推理 / 推论 / 推断 / 推荐 / 推动 全部点亮

学了一个 「量」 →
  量化 / 增量 / 考量 / 测量 全部点亮
```

中文母语者读到这里又会觉得废话。但学员看到这里会觉得 「我可以学得动」 —— 因为不再是 「100 个独立词」 而是 「20 个砖头组合出 100 个词」。

## 三、海外学员对简体 vs 繁体的处理

我们告诉学员:**ModelScope、DeepSeek、Qwen、知乎、CSDN 都是简体,放心去看。** 繁体出现在台湾、香港、古文献,跟他们想读的 MLOps 资料几乎完全不重叠,所以他直接忽略。

这一步是 1 分钟讲完。但如果不讲,学员可能会因为偶尔在网上看到繁体而不安。

---

## 四、拼音是什么 —— 海外学员的「Pleco 输入法」定位

我们对学员的核心论点:**拼音对你不是发音课,是字典输入法。**

具体来讲:
- 看到 「模型」 不会读 → 输入 `mo xing` 进 Pleco → 出来意思 = 模型 / model
- 看到 「推理」 不会读 → 输入 `tui li` 进 Pleco → 出来意思 = inference

学员只需要会**读拼音表**,**不需要发音准**。我们让他们用 HelloChinese 第一周过一遍拼音,然后扔掉。

中文母语者读到这里可能会皱眉:「拼音怎么能不发音?」 —— 答案是:**他不是要说,是要读**。完全不一样的目标。

---

## 五、声调:不是发音,是「输入法选键」

同样的逻辑。我们让学员认得:

```
ā  (1 声) — 平
á  (2 声) — 升
ǎ  (3 声) — 降升
à  (4 声) — 降
a  (轻声) — 轻
```

为什么?因为 Pleco 输入拼音时会要求他们选声调,如果他们不认得声调标记,查字会卡住。

**他们不需要听懂声调,不需要发对声调。** 这是中文母语者最难接受的设定 —— 「你怎么能不学声调?」 —— 但对一个只想读模型卡的人,声调真的就是字典输入界面的一个键。

## 六、海外学员看到的「中文 vs 母语」对照表

韩语母语者:

| | 韩语 | 中文 |
|--|------|------|
| 语序 | SOV(나는 + 모델을 + 훈련) | SVO(我 + 训练 + 模型) |
| 动词变化 | 有 (먹다 → 먹어요 → 먹었다) | 无 (吃 不变) |
| 助词 | 有 (는/을/이/가) | 无 |
| 汉字词 | 70% 是汉字词 → **巨大优势** | 100% 汉字 |

英语母语者:

| | 英文 | 中文 |
|--|------|------|
| 语序 | SVO | SVO ✓ |
| 动词变化 | 有 (-ed, -ing, -s) | 无 |
| 冠词 | 有 (a/an/the) | 无 |
| 汉字词共享 | 无 | — |

**关键观察:**英文学员在语序上很轻松,但在汉字上没有任何起跑线优势。所以他们的总学习时间(6-8 个月)比韩语学员(4 个月)长。

---

## 七、海外学员看到的「韩语母语者捷径」

我们告诉海外学员的事实:**韩语里 70% 是汉字词。** 韩语母语者看到 「部署」 直接对应韩语汉字音 「부서」,意思 90% 命中。这是他从小学到大默默积累的资本,英文学员没有。

| 中文 | 韩语汉字音 | 意思 |
|------|----------|------|
| 模型 | 모형 | model ✓ |
| 训练 | 훈련 | training ✓ |
| 部署 | 부서 | deployment ✓ |
| 量化 | 양화 | quantization ✓ |

这件事对中文母语者只是 「啊韩语跟中文有汉字词的事实」,但对学员来说,这是他选择 4 个月路径(韩语)还是 6-8 个月路径(英语)的关键参数。

### 假朋友陷阱

韩语学员要专门记几个意思已经分化的汉字组:

| 中文 | 韩语汉字音 | 韩语原意(可能误读) | 中文实际意思 |
|------|----------|-------------------|--------------|
| 汽车 | 기차 | 기차 = 火车 | 汽车 = 自动车 |
| 工夫 | 공부 | 공부 = 学习 | 工夫 = 时间/精力 |
| 新闻 | 신문 | 신문 = 报纸 | 新闻 = news |
| 告诉 | 고소 | 고소 = 起诉 | 告诉 = 跟你说 |

中文母语者看到 「告诉」 怎么会觉得 「起诉」?但韩语学员的母语干扰真的会拐到那条路上。请理解。

## 八、海外学员的「第一个 10 个汉字分解」

我们给海外学员的第一个真正的练习,是把 10 个 MLOps 高频汉字拆给他们看:

| 汉字 | 部首 | 我们告诉学员的拆法 |
|------|------|-------------------|
| 模 | 木 | 木 + 莫,木做的样板 |
| 型 | 土 | 土 + 刑,泥塑的形状 |
| 推 | 扌 | 手 + 隹,用手推 |
| 理 | 王 | 玉 + 里,玉的纹路 → 道理 |
| 训 | 讠 | 言 + 川,用话流过 → 教导 |
| 练 | 纟 | 丝 + 柬,整理丝线 → 练习 |
| 部 | 阝 | 立 + 阝,分区域 |
| 署 | 罒 | 网 + 者,布置 |
| 量 | 里 | 旦 + 里,度量单位 |
| 化 | 亻 | 人 + 匕,变化 |

中文母语者读这一段会觉得有点过于幼稚。但对学员来说,这是他第一次理解 「啊汉字是真的有逻辑的」。

然后我们让学员看 10 个汉字怎么组成 5 个领域词:

```
模 + 型 = 模型 (model)
推 + 理 = 推理 (inference)
训 + 练 = 训练 (training)
部 + 署 = 部署 (deployment)
量 + 化 = 量化 (quantization)
```

到这一步学员的反应大致是 「等等,我学了 10 个字 = 我会读 5 个 MLOps 词?那 100 个字 = 50 个词?」 —— 是的,而且实际上数字更夸张,因为这 10 个字会出现在更多组合里。

## 九、给中文母语朋友的三件事

### 1. 「这一切对你来说都是常识」 = 你的优势,也是教学上的盲点

如果你尝试帮一个海外朋友学中文,你会下意识跳过你认为是常识的部分:「啊汉字嘛,你看着写就行」「啊声调,大概有这几个嘛」。这种跳过对学员是致命的。**他需要的恰恰是你跳过的那 5 个观念**。

如果他卡在某个看起来很简单的问题上(比如 「为什么这个动词没变化?」),不是他笨,是他第一次接触没有动词变化的语言。请慢一点。

### 2. 你写的每一篇技术文,正在被零基础学员用作教材

学员不读传统教材。他们读 你 写的 model card、 你 写的知乎答案、 你 写的 CSDN 博客。他们读这些是因为他们要的是真实信息,不是练习题。

所以请你:
- 段落短一点(学员一段读 5-10 分钟,长段他读不完)
- 术语稳定(同一篇里别从 「推理」 切到 「inference」 切到 「推断」)
- 句型清晰(用 「基于」 「用于」 「对...进行」,别太花哨)

中文母语者写技术文时不会刻意做这些,所以多数情况下质量已经够好。但偶尔遇到 「啊我这段写得有点散」 时,记得有海外学员要读你这段。

### 3. 反向应用 —— 中文母语者怎么用同一个 0 阶段方法读懂英文一手资料

如果你自己想快速读 PyTorch / vLLM / Hacker News,这个 0 阶段框架可以反过来用:

| 海外学员的 0 阶段 | 中文母语者读英文的 0 阶段 |
|------------------|---------------------------|
| 学 「汉字是什么」 | 学 「英文词根/前缀/后缀」 |
| 学 「部首」 | 学 「-tion / -ment / -ize / -ation」 |
| 学 「拼音 = 字典输入法」 | 学 「IPA = 字典输入法,不需要会发」 |
| 学 「忽略声调发音」 | 学 「忽略英文重音规则,看字串就够」 |
| 学 「中文是 SVO」 | 学 「英文也是 SVO,你已经会了」 |

你不需要从 「How are you」 开始学英文。直接从 `inference / deployment / quantization` 这些词根开始。然后你会发现,英文 README 的 80% 用 12-15 个固定句型(passive、it is...that、given that、by means of、对应中文的 「被」「是...的」「如果...就」「通过」),只要攻这 15 个就够。

时间节奏跟海外学员一样:4-6 个月可以读懂大部分 PyTorch / vLLM / arXiv 一手资料。
