# 学习指南 04 — 海外学员的模型卡精读流程

> 第四讲(视频)的延伸阅读。给中文母语者看的版本。
> 假设你已经看过 [guide-00](guide-00-foundations.md)、[guide-02](guide-02-vocabulary.md)、[guide-03](guide-03-grammar.md) —— 这三份分别讲了海外学员怎么处理汉字基础、词汇、语法。
> 这一份是**总集篇**:海外学员把前面学的所有东西合在一起,真的去精读一篇模型卡时,他大脑里发生什么。
> 通读约 30-40 分钟。

---

## 一、海外学员到这一阶段已具备的能力

学员到这一关,已经积累了:

1. **220 个领域词** —— 模型卡里出现的核心高频词都背完了
2. **15 个核心句型** —— 「把/将」「用于」「基于」「对...进行」「通过」 等已经能识别
3. **汉字基础直觉** —— 看到陌生词能拆成偏旁,猜个八九不离十
4. (韩语学员)汉字音映射 / (英语学员)字根积木

下一步要做的事,中文母语者读到这里大概会觉得 「啊就是读模型卡嘛」。但海外学员的精读流程是一个非常具体的 5 步动作集合,值得给你看一下。

---

## 二、海外学员认识的「模型卡 5 段结构」

中文母语者读模型卡时不会刻意意识 「这是哪一段」 —— 你的大脑直接抓信息。但海外学员被训练成第一眼就要识别段落结构:

| 段 | 中文标题 | 段里他们要找的信息 |
|----|---------|---------------------|
| 1 | 模型简介 / 简介 | 模型类型、架构、用途 |
| 2 | 模型规格 / 参数说明 | 参数量、上下文长度、量化选项 |
| 3 | 使用方法 / 快速开始 | 安装命令、Python 代码 |
| 4 | 评测结果 / 性能 | 表格 |
| 5 | 许可证 / 使用协议 | 商用是否允许 |

如果你写的模型卡**这 5 个段标题缺一个**,海外学员会卡很久 —— 因为他们的解析依赖这 5 个锚点定位信息。**给中文母语朋友的第一个具体建议**:写模型卡时保留这 5 段标题(或它们的标准变体),不要为了文笔活泼省略掉。

---

## 三、海外学员靠定型句解析每一段

学员不是逐字读的,他们靠 「定型句模式匹配」。下面这些句型只要你写中文模型卡用了,他们就能秒识别:

### 简介段

```
{模型名} 是基于 {架构} 的 {模型类型},支持 ... 等任务。
```

中文母语者看这句只觉得 「正常的开头」,但海外学员看到的是:`是基于...的` + `支持...等任务` 两个模板被同时点亮。学员脑内秒解析。

### 规格段

```
总参数量为 X 亿
激活参数为 Y 亿
支持上下文长度 Z
```

数字+量词+方位词,顺序固定。**只要顺序固定**,海外学员秒读。但如果你写成 「我们这款 Y 亿激活参数,而总参数量是 X 亿」 —— 中文读者读得通,但学员的模板匹配失败。

### 使用段

```
首先,...
然后,...
最后,...
```

学员一眼就知道这是步骤说明。但如果你写成 「我们先做 X,接着做 Y,最终做 Z」 —— 学员需要重新解析,因为 「接着」、「最终」 不在他学的核心 15 句型里。

### 评测段

表格。学员只需要识别表头几个词(模型 / 评分 / 准确率 / 召回率)。

### 协议段

```
本模型采用 {协议} 协议发布。
商用 {允许/禁止}。
```

定型句两条。学员秒过。

## 四、海外学员真实精读 5 段的内部过程

韩文版和英文版的 `guide-04-model-card.md` 给学员准备了 5 个虚构练习段(分别模仿 Qwen / DeepSeek / ChatGLM / Yi / MiniMax 的常见写法)。每一段的精读过程是:

```
[第 1 步] 找出 220 词里的领域词(通常一句里有 3-5 个)
[第 2 步] 用韩语汉字音 / 英语字根猜剩下的词
[第 3 步] 识别 15 个句型里出现的几个
[第 4 步] Pleco 查剩下真不会的词(通常一句里 1-2 个)
[第 5 步] 拼出整段的英文 / 韩文翻译
```

中文母语者读一段大概 5 秒。海外学员各 20-30 分钟。这个 「5 秒 vs 30 分钟」 的差距,正是这门课在压缩的东西。

### 学员遇到的 5 段难点(给你参考)

中文母语者你可能没意识到这些是难点,但学员一定卡:

1. **「亿」 vs 「B」** —— `2350 亿` 是 235B 不是 235M,英语学员经常搞混
2. **复合句型 「通过对X进行Y」** —— 一段里出现 1-2 次,学员要拆 4 个零件才能解析
3. **格式词 「若」「您」「如下」「若干」** —— 比 「如果」「你」「下面」「一些」 更书面,学员第一次见会查字典
4. **「该模型」 vs 「这个模型」 vs 「本模型」** —— 中文母语者无意识切换,但学员会把 `该 / 这个 / 本` 各自当成独立的词学
5. **量词 「一款」 「一种」 「一类」** —— 中文母语者下意识用,学员需要专门学

如果你写模型卡时**尽量统一用法**(比如全篇用 「该模型」,不要中途切到 「本模型」 再切到 「这个模型」),学员的认知负担会减半。

## 五、给中文母语朋友的实战建议

### 1. 写模型卡的 5 个 「友好度」 调整

如果你写中文模型卡,30 秒的额外注意能让海外读者的工作量减半:

(a) **保留 5 段标题**:`简介` / `规格` / `使用方法` / `评测` / `许可证` 或它们的标准变体。不要省略,不要替换成花哨表达。

(b) **术语一致**:同一篇里 「推理」 就一直 「推理」,不要中途切到 「inference」 或 「推断」。

(c) **使用核心 15 句型**:用 「基于」「用于」「采用」「对...进行」「通过」 这种结构,不要写成 「咱们这模型骨子里其实就是 Transformer」 之类口语化表达。

(d) **段落短**:学员一段读 5-10 分钟,过长段他读不完。一段 5-7 句以内最好。

(e) **数字单位标注**:`2350 亿(235B)` 这样英中并列写,学员秒解析。如果你只写 `2350 亿`,英语学员会想很久。

### 2. 处理海外开发者提的中文 Issue 时

如果你收到一个海外开发者用中文发的 Issue,语气有点僵 —— 比如全文都是 「该项目的 X 模块在执行 Y 操作时出现错误」 —— 不要笑。那是他用 15 句型精确堆叠出来的。你回复一句中文(正常的中文,不需要特意简化),他会很感激。**他写中文是用 4 个月学习换来的**,你的中文回复对他是真实的语言练习材料。

### 3. 如果他用磕巴的中文问问题,你怎么回答最有价值

最好的回答方式是:**用清晰的中文回答 + 关键术语标注英文**。

例如学员问:「请问该模型是否支持 fp8 推理?」

不太好的回答:「支持的呀,你试试 q.fp8.bin 那个。」(口语化、术语没注英文,学员可能要查 「试试」 是什么意思)

好的回答:「是的,该模型支持 fp8 推理。请下载 q.fp8.bin 文件并使用 vLLM 0.5+(推理引擎,inference engine)加载。」(清晰句型 + 术语英文标注)

第二种回答让学员能立即用,而且他从你的中文里学到一个新句型 + 几个新术语。

### 4. 反向应用 —— 中文母语者用同样思路精读英文一手资料

如果你想反过来快速读 PyTorch / vLLM / Hacker News:

```
英文 5 段结构(对应中文模型卡):
  Introduction (简介)     —— 模型介绍
  Specifications (规格)   —— 参数
  Usage (使用)            —— 代码示例
  Benchmarks (评测)       —— 性能比较
  License (协议)          —— 许可证
```

英文 README 的 80% 用 12-15 个固定句型(passive、it is...that、given that、provided that、in order to、by means of)。攻这 15 个英文句型 + 200 个英文领域词根,4-6 个月你也能跟海外学员一样,精读 PyTorch / vLLM / arXiv 一手资料。

## 六、海外学员的毕业产出

毕业条件 2 个:

1. **一个真实的中文 GitHub Issue**(到 vLLM / ms-swift / FastChat / ChatGLM / 或其他中文开源项目)
2. **一篇 300 字以上的中文知乎回答**

第一次写完肯定磕磕巴巴。但写完发出去这件事本身就是毕业,不需要等回复。

如果你看到一个海外开发者发的第一个磕磕巴巴的中文 Issue —— **那是他 16 周路径的最后一步**。请你温柔地接住它。

---

## 七、海外学员毕业后的下一步

毕业不等于结束。海外学员毕业后的标准路径:

1. 每天读一段中文一手资料(模型卡 / 知乎 / CSDN),保持节奏
2. 提交第二个 OSS 贡献,这次是 PR 而不是 Issue,中文 PR 描述
3. 扩展到自己的子领域(自动驾驶、机器人、多模态、语音等)
4. 可选:用同样方法学日语(汉字资本可以迁移)

---

## 八、整门课的最后一句话

这是这门课最后一份学习指南。海外学员到了这一关,已经具备读你写的中文 + 写出能用的中文的能力。中文母语者读到这一关,知道了 「我们写的中文,在被谁读、怎么读、读多久」。

谢谢中文母语朋友陪海外学员走完这 4 个月。你的写作就是教材,你的回应就是最好的反馈。
