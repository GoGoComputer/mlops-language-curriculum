# 학습 가이드 00 — 중국어를 한 번도 본 적 없는 사람을 위한 0단계

> 이 가이드는 중국어를 한 번도 학습한 적 없는 분이 학습 가이드 01부터 시작하기 전에 한 번 통독하는 자료입니다.
> 외울 건 없습니다. 30~45분 통독하면서 "아, 중국어는 이런 언어구나"라는 직관을 만드는 게 목표입니다.
> 다른 가이드(01-04)는 모두 이 가이드의 내용을 안다는 가정 위에서 시작합니다.

## 1. 시작하기 전에 — 우리가 풀어야 할 5가지 질문

이 0단계를 다 읽고 나면 다음 5가지에 답할 수 있게 됩니다.

1. 중국어는 한국어와 어떻게 다른 언어인가?
2. 한자(漢字)는 어떻게 의미를 가지는 글자인가?
3. 병음(拼音)이란 무엇이고, 우리는 그걸 왜 쓰는가?
4. 사성(四声)은 외워야 하는가?
5. 왜 한국어 화자가 중국어 독해에서 유리한가?

지금 답할 수 없어도 됩니다. 끝까지 읽으면 자연스럽게 답이 잡혀요.

## 2. 중국어는 어떻게 생긴 언어인가요?

먼저 한국어와 비교해봅시다.

한국어로 "나는 모델을 훈련시킨다"라고 쓰면:

```
나   는    모델  을    훈련시킨다
주어 조사  목적어 조사  동사
```

조사("는", "을")가 있고, 동사가 마지막에 옵니다. 그리고 동사 "훈련시키다"는 시제·서법에 따라 활용됩니다 — "훈련시켰다", "훈련시키고 있다", "훈련시킬 것이다"...

같은 문장을 중국어로 쓰면:

```
我      训练         模型
워      쉰리엔        모싱
나      훈련하다     모델
```

세 단어. 조사 없습니다. 동사가 가운데에 옵니다 (영어와 같은 SVO). 그리고 가장 충격적인 점 — 동사 `训练`(훈련하다)은 시제가 바뀌어도 **모양이 변하지 않습니다**. 시제는 별도 보조 단어로 표시해요.

| 의미 | 중국어 |
|------|--------|
| 나는 훈련시킨다 | 我训练 |
| 나는 훈련시켰다 | 我训练**了** |
| 나는 훈련시키고 있다 | 我**在**训练 |
| 나는 훈련시켜본 적 있다 | 我训练**过** |
| 나는 훈련시킬 것이다 | 我**将**训练 |

`训练` 부분은 그대로고, `了 / 在 / 过 / 将` 같은 보조 단어가 시제를 만듭니다.

### 한국어와 가장 다른 점 3가지

1. **글자 체계가 다르다.** 한국어는 한글 + 가끔 한자. 중국어는 한자만.
2. **동사가 활용되지 않는다.** "먹다 → 먹어요 → 먹었다" 같은 변화가 중국어에는 없음.
3. **어순이 SVO다.** 한국어 SOV(주어-목적어-동사)와 다르지만, 영어와는 같음.

### 한국어와 친숙한 점 1가지

중국어 단어의 상당수가 한국어 한자어와 같은 한자로 만들어져 있습니다.

| 중국어 | 한국 한자어 | 의미 |
|--------|------------|------|
| 模型 | 모형 | 모델 |
| 训练 | 훈련 | training |
| 部署 | 부서 | 배포 |
| 量化 | 양화 | 양자화 |
| 推理 | 추리 | 추론 |

이게 한국어 화자가 중국어 도메인 자료를 읽을 때 가지는 가장 큰 자산입니다. 영어 화자는 이 자산이 없어요.

## 3. 한자(漢字)란 무엇인가요?

한자는 **표의문자(表意文字)**, 즉 "의미를 나타내는 글자"입니다. 한글의 "ㄱ ㄴ ㄷ"이나 영어의 "a b c"는 소리를 나타내는 표음문자죠. 한자는 그게 아니라 **글자 하나가 의미 한 단위**입니다.

가장 직관적인 예:

```
木  나무 한 그루
林  나무 두 그루 = 수풀
森  나무 세 그루 = 빽빽한 숲
```

진짜로 그래요. 木은 나무 한 그루의 모양이 추상화된 거고, 林은 木 두 개, 森은 木 세 개. 의미가 시각적으로 들어가있죠.

다른 예:

```
日  태양 (해의 모양)
月  달 (달의 모양)
明  日 + 月 = 밝다 (해와 달이 함께 있으니)
```

한자는 처음에는 그림이었어요. 시간이 지나면서 추상화되었지만, **의미 힌트는 글자 안에 보존되어 있습니다.**

### 부수(部首) — 한자의 의미 부품

복잡한 한자도 더 작은 의미 단위로 분해됩니다. 그게 **부수**예요. 부수는 보통 한자의 왼쪽이나 위에 있고, **의미 카테고리를 알려줍니다.**

예:

```
推 (밀다)  =  扌 (손)  +  隹 (새)
                ↑
            손 부수가 있으니 "손과 관련된 동작"
```

```
流 (흐르다)  =  氵 (물 세 방울)  +  나머지
                  ↑
              물 부수가 있으니 "액체와 관련"
```

```
烧 (태우다)  =  火 (불)  +  나머지
                  ↑
              불 부수가 있으니 "열·연소 관련"
```

```
想 (생각하다)  =  심장 모양 + 마음 心
                          ↑
                      "마음의 작용"
```

자주 보는 부수 10개를 정리해봤습니다. **외우려고 하지 마세요.** 자료에서 자주 보면서 자연스럽게 인식하게 됩니다.

| 부수 | 의미 카테고리 | 들어가는 한자 예 |
|------|--------------|------------------|
| 扌 | 손 | 推, 拉(당기다), 打(때리다), 接(받다) |
| 氵 | 물 | 流, 海(바다), 河(강), 湖(호수) |
| 火 | 불 | 烧, 炒(볶다), 烤(굽다) |
| 心 / 忄 | 마음 | 想, 思(생각), 慢(느리다) |
| 木 | 나무 | 树(나무), 林, 森, 板(판자) |
| 口 | 입 | 吃(먹다), 喝(마시다), 听(듣다), 说(말하다) |
| 亻 | 사람 | 你(너), 他(그), 们(들) |
| 女 | 여자 | 好(좋다), 妈(엄마), 她(그녀) |
| 言 / 讠 | 말 | 说, 话(말씀), 训(가르치다), 词(낱말) |
| 阝 | 언덕/도시 | 部(부서), 都(도시), 院(원) |

**핵심 통찰**: 한자 한 글자를 학습한다는 건 그 글자가 들어가는 수십 개의 단어를 한 번에 점화한다는 뜻입니다. `扌`(손) 부수를 인식할 수 있게 되는 순간, `推 / 拉 / 打 / 接` 모두 "어떤 손동작인가?"라는 단서가 자동으로 붙어요.

## 4. 간체자 vs 번체자 — 둘 중 어느 걸 학습?

옛날에 쓰이던 한자, 우리가 가끔 한국어 신문에서 마주치는 한자는 획수가 많은 **번체자(繁體字)**예요. 1950년대 중국 본토에서 한자를 단순화했고, 그게 **간체자(简体字)**입니다.

| 번체 | 간체 | 의미 |
|------|------|------|
| 馬 | 马 | 말 (동물) |
| 學 | 学 | 배움 |
| 開 | 开 | 열다 |
| 國 | 国 | 나라 |
| 機 | 机 | 기계 |
| 龍 | 龙 | 용 |
| 體 | 体 | 몸 |

같은 글자입니다. 그냥 모양이 단순화된 것뿐.

**ModelScope, DeepSeek, Qwen, Zhihu, CSDN — 모두 간체자.** MLOps 자료의 99% 이상이 간체. 이 커리큘럼도 간체만 다룹니다. 번체자는 무시하셔도 됩니다.

(대만/홍콩은 여전히 번체. 하지만 우리가 읽고 싶은 자료는 다 본토 간체에 있어요.)

## 5. 병음(拼音)이란 무엇인가요?

병음은 **한자의 발음을 로마자로 적은 것**입니다. 한국으로 치면 "안녕하세요"를 "annyeonghaseyo"라고 적은 것과 비슷.

예시:

```
模型  →  mó xíng    (모-싱)
推理  →  tuī lǐ      (투이-리)
部署  →  bù shǔ      (부-슈)
```

### 병음을 왜 쓰나요?

우리에게 병음은 두 가지 용도로만 씁니다:

1. **사전(Pleco)에서 한자를 검색할 때 입력**
2. **모르는 한자를 만났을 때 대략적인 음 추측**

**우리는 발음을 정확하게 할 필요 없습니다.** 회화 학습이 아니니까요.

### 병음 음절 구조 (간단)

병음 한 음절 = **자음(声母)** + **모음(韵母)** + **사성(声调)** 표시

예: `mā`

```
m   +   a   +   ā(1성 표시)
자음     모음    톤
```

자음은 영어와 비슷한 게 많지만, 일부는 다릅니다:

| 병음 | 발음 힌트 |
|------|-----------|
| zh, ch, sh | 영어의 j, ch, sh와 비슷하나 혀를 약간 말아서 |
| z, c, s | "츠"에 가까운 z, c, s |
| q | 영어 "ch"에 가까움 |
| x | 영어 "sh"에 가까움 |
| r | 한국어 ㄹ과 영어 r 사이 |

이걸 정확히 발음하려면 듣기/말하기 훈련이 필요한데, 우리는 그 부분을 의도적으로 제외했어요.

**필요한 능력**: 병음을 보고 "아 대략 이렇게 발음하나보다" 정도. Pleco에 입력할 수 있을 정도.

**필요 없는 능력**: 정확한 발음.

## 6. 사성(四声) — 외우지 말고 인식만

중국어는 **성조 언어**입니다. 같은 음절이라도 톤(높낮이)에 따라 의미가 달라져요.

같은 `ma`라도:

| 성조 | 표기 | 한자 | 의미 |
|------|------|------|------|
| 1성 | mā | 妈 | 엄마 |
| 2성 | má | 麻 | 마(식물) |
| 3성 | mǎ | 马 | 말(동물) |
| 4성 | mà | 骂 | 욕하다 |
| 경성 | ma | 吗 | (의문조사) |

성조 표시 모양으로 톤을 식별:

| 표기 | 모양 | 발음 이미지 |
|------|------|-------------|
| ā (1성) | 평탄 | "아—" 한 톤으로 길게 |
| á (2성) | 올라감 | "아?" 끝을 올리듯 |
| ǎ (3성) | 내려갔다 올라옴 | "아아아" 가운데가 낮게 |
| à (4성) | 내려감 | "아!" 끊는 듯이 |
| a (경성) | 가볍게 | 끝에 짧게 붙는 톤 |

### 우리는 사성을 어떻게 다루나요?

**필요**: 병음 옆의 성조 표시를 읽을 수 있어야 함. Pleco에 입력할 수 있어야 함.

**불필요**: 정확하게 발음하기. 듣고 구별하기.

성조를 듣고 구별하는 능력을 "성조 청해(聽解)"라고 하는데, 우리는 청해 학습을 통째로 제외했어요. 회화하지 않을 거니까요.

## 7. 한국어와 중국어의 차이 정리

| 항목 | 한국어 | 중국어 |
|------|--------|--------|
| 글자 체계 | 한글 + 한자 | 한자만 (간체) |
| 어순 | SOV (나 + 모델 + 훈련) | SVO (我 + 训练 + 模型) |
| 동사 활용 | 있음 (먹다 → 먹어요) | 없음 (吃 그대로) |
| 시제 표시 | 어미 변화 | 보조 단어 (了, 过, 将, 在) |
| 조사 | 있음 (는/을/이/가) | 없음 또는 어순으로 |
| 발음 | 음절 + 받침 | 음절 + 사성 |
| 표의/표음 | 표음 (한글) | 표의 (한자) |

**가장 어색한 부분**: 활용형 없음 + 조사 없음. "我训练模型"에 "는/을"이 없어서 처음에는 어색합니다. 일주일 정도 노출되면 자연스러워져요.

## 8. 한국어 화자의 강점 — 한자어 직관

한국어의 약 70%가 한자어입니다. 그리고 그 한자어는 중국어와 같은 한자로 만들어진 경우가 많아요.

비교 예:

| 중국어 | 한자 그대로 한국 한자음 | 의미 |
|--------|----------------------|------|
| 模型 | 모형 | 모델 ✓ |
| 训练 | 훈련 | training ✓ |
| 部署 | 부서 | 배포 ✓ |
| 量化 | 양화 | quantization ✓ |
| 微调 | 미조 | fine-tuning ✓ |
| 参数 | 참수 (한국어 참수는 다름. 매개변수) | parameter |
| 推理 | 추리 (한국어 의미는 추론과 비슷) | inference ✓ |

**한국어 한자음을 한 번만 떠올려보면, 의미의 70-90%가 직관으로 잡힙니다.** 영어 화자에게는 없는 자산이에요.

이게 우리 커리큘럼이 4개월로 압축된 핵심 이유예요. 영어 화자는 같은 학습에 6-8개월 필요합니다.

### 함정 단어 — 의미가 갈라진 한자 조합

같은 한자인데 한국어와 중국어에서 의미가 달라진 단어가 있어요. 약 30개 정도. 자료에서 만나면서 자연스럽게 학습됩니다.

| 중국어 | 한국 한자음 | 한국어 의미 (혹시 헷갈릴 만한) | 중국어 실제 의미 |
|--------|------------|-----------------------------|------------------|
| 汽车 | 기차 | 기차 | 자동차 (한국어 기차는 火车) |
| 工夫 | 공부 | 공부 | 시간/노력 (한국어 공부는 学习) |
| 新闻 | 신문 | 신문 | 뉴스 (한국어 신문은 报纸) |
| 告诉 | 고소 | 고소 | 알려주다 |

처음에 헷갈려도 괜찮아요. 함정 단어는 만나면서 자동으로 정정됩니다.

## 9. 첫 한자 10개 — 부수와 의미 분해 시연

이제 실제로 도메인 어휘에서 가장 자주 나오는 한자 10개를 분해해봅시다. 외우는 게 아니라 **분해 흐름을 익히는 게 목표**예요.

| 한자 | 병음 | 부수 | 분해 직관 | 한국 한자음 | 의미 |
|------|------|------|----------|------------|------|
| 模 | mó | 木 (나무) | 木 + 莫 → "나무로 본뜬 형태" | 모 | 본·모형 |
| 型 | xíng | 土 (흙) | 土 + 刑 → "틀에 넣어 만든 형태" | 형 | 형태·타입 |
| 推 | tuī | 扌 (손) | 손 + 隹(새) → "손으로 미는 동작" | 추 | 밀다, 추론하다 |
| 理 | lǐ | 王 (옥) | 王 + 里 → "옥의 결" | 리 | 이치, 다스리다 |
| 训 | xùn | 讠 (말) | 말 + 川(흐르다) → "말로 흐르듯 가르치다" | 훈 | 가르치다 |
| 练 | liàn | 纟 (실) | 실 + 柬 → "실을 정리하다" | 련 | 단련하다, 연습하다 |
| 部 | bù | 阝 (언덕) | 立 + 阝 → "구역을 나누다" | 부 | 부분, 부서 |
| 署 | shǔ | 罒 (그물) | 罒 + 者 → "그물 친 관청" | 서 | 배치하다, 관청 |
| 量 | liàng | 里 (마을) | 旦(해) + 里 → "재는 단위" | 량 | 양, 측정하다 |
| 化 | huà | 亻 (사람) | 사람 + 匕 → "변화" | 화 | 변하다 |

### 조합으로 만들어지는 단어

이 10개를 조합하면 도메인 단어가 만들어집니다.

```
模 + 型  =  模型 (모형)        →  모델
推 + 理  =  推理 (추리)        →  추론, inference
训 + 练  =  训练 (훈련)        →  training
部 + 署  =  部署 (부서)        →  배포, deployment
量 + 化  =  量化 (양화)        →  quantization
```

**중요한 통찰**: 한자 100개 정도만 알면 단어 1,000개의 의미를 추측할 수 있어요. 한자 = 빌딩 블록, 단어 = 빌딩 블록의 조합.

---

## 10. 첫 단어 5개 — 처음 만나서 이해하기까지

각 단어를 어떻게 처음 만나서 이해하는지 시연해드릴게요. 자료에서 처음 보는 단어를 만났을 때, 머릿속에서 일어나는 흐름이에요.

### 사례 1: 训练

```
1. 한자 식별:  训 + 练
2. 부수 보기:  讠(말) + 纟(실)
3. 직관:       말로 가르치고 실타래처럼 정리하다
4. 한국 한자음: 훈 + 련 = 훈련
5. 결론:       training ✓
```

### 사례 2: 部署

```
1. 한자 식별:  部 + 署
2. 부수 보기:  阝(언덕) + 罒(그물)
3. 직관:       구역을 나누어 관청에 배치하다
4. 한국 한자음: 부 + 서 = 부서
5. 한국 한자어 "부서"는 "배치"와 의미가 통함
6. 결론:       deployment, 배포 ✓
```

### 사례 3: 量化

```
1. 한자 식별:  量 + 化
2. 부수 보기:  里(마을, 단위) + 亻(사람)
3. 직관:       양으로 변환하다
4. 한국 한자음: 양 + 화 = 양화
5. ML 도메인 지식 적용: "양으로 만든다" + 비트 압축 = 양자화
6. 결론:       quantization ✓
```

### 사례 4: 微调

```
1. 한자 식별:  微 + 调
2. 부수 보기:  彳(걷다) + 讠(말)
3. 직관:       작은 조정
4. 한국 한자음: 미 + 조 = 미조
5. ML 도메인 지식: 작게 조정 = 미세조정
6. 결론:       fine-tuning ✓
```

### 사례 5: 推理

```
1. 한자 식별:  推 + 理
2. 부수 보기:  扌(손) + 王(옥)
3. 직관:       이치를 손으로 밀어 도출
4. 한국 한자음: 추 + 리 = 추리
5. ML 도메인 지식: "추리"는 한국어로 reasoning, 즉 모델 추론
6. 결론:       inference ✓
```

이 다섯 단계 분해 흐름을 약 30번 반복하면, **한 달 후에는 새 단어를 만나도 1-2초 안에 의미를 90% 추측**할 수 있게 됩니다. 그게 가이드 02에서 다룰 220 어휘 학습의 진짜 메커니즘이에요.
