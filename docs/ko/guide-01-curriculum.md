# 학습 가이드 01 — 학습 전략과 4개월 계획

> 강의 01(영상)의 동기부여를 실제 학습 계획으로 옮기는 가이드입니다.
> 학습 시작 전 한 번 통독, 매주 일요일 한 번 더 점검하시는 흐름을 권장합니다.

---

## 0. 시작 전 자기 진단

다음 5문항에 답해보세요.

1. 중국어 학습 경험이 있나요? (네/아니요)
2. 한국어 한자어(推進, 訓練, 部署 등)를 보고 의미를 추측할 수 있나요? (네/아니요)
3. 일주일에 5시간 이상 투자할 수 있나요? (네/아니요)
4. 이 학습의 목적은 회화인가요, 독해인가요? (회화/독해/둘 다)
5. 4개월 후 GitHub Issue를 중국어로 직접 작성할 의향이 있나요? (네/아니요)

| 답변 패턴 | 적용 방안 |
|-----------|-----------|
| 1·2 모두 "네" | 표준 4개월 일정 (이 가이드 그대로) |
| 1 "아니요" + 2 "네" | 표준 4개월. 한자어 직관이 출발 가속 보장 |
| 1 "네" + 2 "아니요" | Month 1 한자 학습을 5주로 연장 |
| 3 "아니요" | 6개월 또는 8개월로 연장 |
| 4 "회화" | 본 커리큘럼 부적합. HelloChinese 등 회화 코스 권장 |
| 5 "아니요" | 졸업 산출물은 동기 유지 핵심. 다시 고려 권장 |

## 2. 일일·주말 루틴

### 평일 45분

```
00:00 ~ 00:10  Anki 한자/어휘 30개 (신규 + 복습)
00:10 ~ 00:35  자료 한 단락 독해 (모델 카드 / Zhihu / CSDN)
00:35 ~ 00:45  막힌 단어 정리 + 다음 날 Anki 추가 + DeepL 비교
```

### 주말 90분

```
00:00 ~ 00:30  Anki 누적 복습 (밀린 카드 처리)
00:30 ~ 01:15  신규 자료 1페이지 통독 (긴 자료 도전)
01:15 ~ 01:30  주간 회고: 적중률, 막힌 패턴, 다음 주 조정
```

## 5. 측정과 피드백

**일일**
- Anki 적중률 (목표 80%+)
- 한 단락 독해 시간 (Week 1: 30분 → Week 16: 5분)

**주간 (일요일)**
- DeepL과 자기 독해 비교 — 차이가 큰 부분이 학습 포인트
- 적중률 70% 미만 → 다음 주 신규 카드 줄이고 복습 강화

**월간**
- 새 자료 1페이지를 DeepL 없이 30분 안에 읽을 수 있는가
- 못한다면 Month 1 어휘를 다시 점검

## 6. 동기부여 유지

학습 슬럼프는 보통 Week 5-6 (Month 2 초반)에 옵니다. 도메인 어휘는 다 외웠는데, 실제 자료에서 막히는 단어가 너무 많이 보이기 시작하는 시점이거든요.

대응 3가지:
1. **친숙한 분야 자료로 갈아타기** — 게임 좋아하면 Stellaris 中文 위키, 사주 좋아하면 中文 명리학 글. 비기술 자료에서 한자 노출량 늘리기
2. **목표 재확인** — 4개월 후 GitHub Issue 1건 게시. 그 1건이 졸업 조건의 전부
3. **첫 단락 리뷰** — 한 달 전 막혔던 단락을 다시 읽어보면 얼마나 빨라졌는지 체감

## 7. 졸업 산출물

**산출물 1: GitHub Issue 1건**
- vLLM, ms-swift, FastChat, ChatGLM 등 중국 OSS 프로젝트에 실제 게시
- 템플릿: `docs/ko/github-issue-template.md`
- 답변을 받지 못해도 OK. 작성·게시 자체가 산출물

**산출물 2: Zhihu 답변 1건 (300자+)**
- 자기 분야 질문 검색 → 답변 작성 → 게시
- 첫 답변이 어색해도 게시 자체가 졸업 조건

---

## 8. 졸업 후 다음 단계

1. Daily reading habit — 매일 모델 카드 1단락 또는 Zhihu 답변 1편
2. 두번째 OSS 기여 — PR로 발전 (코드 + 중국어 PR 설명)
3. 자기 분야 확장 — MLOps → 자율주행, 로봇, 멀티모달 등
4. (선택) 일본어로 확장 — 한자 자산 활용

## FAQ

**Q. 사성을 정확히 발음 못해도 정말 괜찮나요?**
A. 네. 발음은 한자 검색용 보조 수단입니다. Pleco/사전에 병음 입력할 정도면 충분.

**Q. HSK 자격증을 따고 싶은데요?**
A. 본 커리큘럼은 HSK 대비가 아닙니다. HSK는 회화·듣기 비중이 큼. 별도 준비 필요.

**Q. 4개월 후에도 회화는 못 하나요?**
A. 못 합니다. 의도적으로 제외했습니다. 회화는 별도 6-12개월 추가 투자.

**Q. 매일 45분이 부담입니다.**
A. 30분으로 줄여도 OK. 졸업 일정이 5-6개월로 늘어남. 0분만 안 되면 됩니다.

**Q. 어디까지 읽을 수 있게 되나요?**
A. ModelScope/Hugging Face 모델 카드 1차 독해, Zhihu 트러블슈팅 답변 통독, 중국어 GitHub Issue 작성. 학술 논문 본문은 졸업 후 추가 학습 필요.
