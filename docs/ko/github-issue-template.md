# GitHub Issue — 중국어 템플릿 모음

> DeepSeek / Qwen / GLM 등 중국 AI 프로젝트 레포에 Issue·Discussion·PR 작성 시 그대로 복붙해서 사용.
> `[]`로 표시된 부분만 자기 상황에 맞게 채우면 됨.

## 자주 쓰는 한 줄 표현 모음

| 한국어 | 중국어 |
|--------|--------|
| 안녕하세요, ~를 보고하고 싶습니다 | 您好，我想反馈一个问题 |
| ~를 만났습니다 | 我遇到了 ... |
| 재현 가능합니다 | 可以稳定复现 |
| 재현 불가능합니다 | 无法稳定复现 |
| ~를 시도해봤는데 효과 없습니다 | 我尝试了 ... 但没有效果 |
| 도움 부탁드립니다 | 请帮忙看看 |
| 시간 내주셔서 감사합니다 | 感谢您抽时间查看 |
| 추가 정보가 필요하면 알려주세요 | 如需更多信息请告知 |
| 관련 코드는 다음과 같습니다 | 相关代码如下 |
| 같은 문제를 겪고 있습니다 | 我也遇到了同样的问题 |
| +1 (동의) | 同+1 |
| 이 issue는 아직 해결되지 않았습니다 | 这个问题还没解决 |
| 이미 PR을 제출했습니다 | 已提交 PR：#[번호] |
| 다음 버전에서 수정될 예정입니다 | 将在下一版本中修复 |

## 톤 가이드

- 중국 오픈소스 커뮤니티는 **존댓말(您)** 사용이 일반적. `你`보다 `您` 우선.
- 무뚝뚝한 직설보다 `请` `麻烦` `感谢` 같은 완곡어 사용 권장.
- 영어 기술 용어는 **그대로 영어로 써도 무방** (PyTorch, transformer, batch size 등).
- 코드 블록은 영어와 동일하게 ` ``` ` 사용.
- 이모지는 자제. 중국 오픈소스 이슈 트래커는 영문권 대비 절제된 톤.
